multidim.PointCloud.sever¶

PointCloud.sever()[source]¶

Subdivide PointCloud into several smaller PointClouds, using the known 0-dimensional persistence diagram. This is an iterator (it YIELDS the terms).

Two points end up in the same PointCloud if and only if they are connected by a sequence of edges of length < cutoff.

See also

make_pers0()
func:reset

Notes

This uses the 0-dimensional Persistence Diagram of the PointCloud; therefore, you should run self.reset() and self.make_pers0(cutoff) first.

Examples

>>> pc = PointCloud(np.array([[0.,0.],[0.,0.5],[1.,0.],[5.,0.],[6.,0.],[5.,-0.6]]), max_length=-1.0)
>>> pc.make_pers0(cutoff=1.9)
>>> for sub_pc in pc.sever():
...     print(sub_pc)
...     print(sub_pc.coords)
