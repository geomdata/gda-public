multidim.Simplex¶

class multidim.Simplex(cellcomplex, dim, index)[source]¶

This class is a convenient container to access the data in the pd DataFrame stratum[dim] of a SimplicialComplex. It is always faster to access stratum[dim].loc[index] directly.

Parameters:

cellcomplex : SimplicialComplex

The SimplicialComplex to which this Simplex belongs.

dim : int

The dimension in which this Simplex lives.

index : int

The abstract index of this Simplex.

See also

SimplicialComplex
A class for abstract simpicial cell complexes.

Notes

A finite (abstract) simplicial complex is a finite set \(A\) together with collection \(\Delta\) of subsets of \(A\) such that if \(X \in \Delta\) and \(Y \subset X\) then \(Y \in \Delta\). [R6]

References

[R6](1, 2) D. Feichtner-Kozlov, Combinatorial algebraic topology. Berlin: Springer, 2008.
